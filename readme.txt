Best WordPress Applause Button & Rating Plugin. Add an Applause Button to your website and let users like your content by "clapping".

=== Description ===

**Add an "Applause Button" to your website and let users like your content by "clapping". This plugin is also generating rich snippet ratings which will show up as 5 star rating in the searchengine results. Make your website stick out by using this plugin!**


== Key Features ==
* Very lightweight, loads fast.
* Responsive and customizable shortcodes.
* Insert shortcode directly from the post editor.
* Rich Snippet 5 star rating.


== Shortcodes: ==

[cp-applause-button] default button

[cp-applause-button style="width:100px;height:100px;"] changed size

[cp-applause-button color="red"] changed color

[cp-applause-button multiclap="true"] multiple claps (by default it true)

Custom shortcode example:
[cp-applause-button color="red" style="width:100px;height:100px;" multiclap="true"]
